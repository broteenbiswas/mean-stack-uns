# YOU N SOCIETY(Internal app)

## Synopsis

An internal app for volunteers to see their assigned programs and no of kids they are impacting currently.

### Functionalities:

* Volunteer Registration
* Admin Login
* Admin Dashboard
* Volunteer Query
